The following examples demonstrate syncing and building firmware sources using
the `Qualcomm_Linux.SPF.2.0|AP|Standard|OEM|NoModem` distribution. Modify the
instructions based on your access profile.

```bash
mkdir -p <FIRMWARE_ROOT>
cd <FIRMWARE_ROOT>
git clone -b <firmware release tag> --depth 1 https://qpm-git.qualcomm.com/home2/git/qualcomm/qualcomm-linux-spf-2-0_ap_standard_oem_nomodem.git
```

The `git clone` command clones the content into the
`<FIRMWARE_ROOT>/qualcomm-linux-spf-2-0_ap_standard_oem_nm` directory.
