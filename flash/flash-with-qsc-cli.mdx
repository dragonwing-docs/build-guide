#### Flash a software image

- Identify the `--serialnumber` parameter for your device
  ```bash
  pcat -devices
  ```

  ```text Sample output
  Searching devices in Device Manager, please wait for a moment…
  ID | DEVICE TYPE | DEVICE STATE | SERIAL NUMBER | ADB SERIAL NUMBER | DESCRIPTION
  NA | NA          | EDL          | BE116704      | be116704          | Qualcomm USB Composite Device:QUSB_BULK_CID:042F_SN:BE116704
  ```

- Identify the `--buildflavor` for the image to be flashed. To list the build flavors, run the following command on the host computer
  ```bash
  qsc-cli chip-software flash --workspace-path "<absolute-workspace-path>" --list-buildflavor
  ```

- Flash the image
  ```bash
  qsc-cli chip-software flash --workspace-path "<absolute-workspace-path>" --buildflavor "<build-flavor>" --serialnumber "<serial-number>"
  ```
