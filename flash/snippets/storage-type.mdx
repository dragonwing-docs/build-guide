Storage configuration is predefined for each chipset and reference platform,
with a storage type (UFS or eMMC) already set by default. If you need to modify
the storage configuration, you must rebuild the image from source with the required
settings, as this cannot be changed directly in the prebuilt binaries.

| Chipset / Platform | Default Storage Type  |
|--------------------|-----------------------|
| QCS6490            | UFS                   |
| IQ‑9075 EVK        | UFS                   |
| IQ‑8275 EVK        | UFS                   |
| IQ‑615 EVK         | eMMC                  |
